# AustenMovies - uma homenagem à minh autora favorita

# Características principais:
    - Todo o conceito do site foi alterado, mantendo apenas o esqueleto e funções;
    - Carousel;
    - Uso do Materialize;
    - Botão para trailer;
    - Dos títulos dos filmes, apenas Pride & Prejudice e Emma seguem o layout original, todos os outros foram construídos em um gerador de Imagem com transparência, dado que não consegui encontrá-los gratuitos;

# O que precisa melhorar no futuro:
    - Utilizar API TMDB para buscar em cada filme:
        - trailers;
        - banners;
        - ano de lançamento;
        - classificação indicativa;
        - tempo de duração;
        - gênero;
    - Refazer todo o layout e organizá-lo usando alguma biblioteca de CSS para que fique mais moderno e responsivo;
    - Refazer navbar de acordo com modelos mais modernos e usando biblioteca de CSS;
    - Atualizar os tamanhos de px para rem/em/vh;
